# TCG-SWE-Primer
Software engineering primer for new Triton Consulting Group analysts.

To start the backend:
- Add a `.env` file and fill it out with your information based on the `.env.example` (only fill out the missing Supabase information)
- In your terminal, type `npm start` and return.

To start the frontend:=
- Add a `.env` file that is an exact copy of the `.env.example`
- In your terminal, type `npm run dev` and return.
